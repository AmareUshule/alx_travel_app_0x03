## Objective
Implement CRUD APIs for Listings and Bookings using Django REST Framework and document endpoints.

## Endpoints

### Listings API
- GET /api/listings/
- POST /api/listings/
- GET /api/listings/{id}/
- PUT /api/listings/{id}/
- DELETE /api/listings/{id}/

### Bookings API
- GET /api/bookings/
- POST /api/bookings/
- GET /api/bookings/{id}/
- PUT /api/bookings/{id}/
- DELETE /api/bookings/{id}/

## Tools Used
- Django
- Django REST Framework
- Postman

## Testing
